# Saathvik's Portfolio (Terminal Edition)

A unique, highly interactive personal portfolio website blending a **Michael Jackson "This Is It" Cinematic Style** with a **Linux/Hacker Terminal** interface.

## 🌟 Live Features

-   **Cinematic Intro**: Gold & Black animated typography sequence.
-   **Terminal OS**: Fully functional command-line interface.
    -   `projects`: Lists AI & Web projects (Honey, PlayHub, Lexis).
    -   `skills`: Summons an interactive 3D 360° rotating skill globe.
    -   `education`: Displays academic background (IIT Jodhpur, LST).
-   **Visuals**:
    -   Theme: **Gold (#FFD700)**, **Black (#000)**, **Silver**.
    -   Font: Cinzel (Cinematic) & Fira Code (Terminal).
    -   **Glassmorphism**: Translucent terminal window.

## 🚀 Getting Started

1.  **Clone the repo**:
    ```bash
    git clone https://github.com/broskell/portfolio-terminal.git
    cd portfolio-terminal
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    # Also requires TagCloud
    npm install TagCloud
    ```

3.  **Run locally**:
    ```bash
    npm run dev
    ```

4.  **Build for production**:
    ```bash
    npm run build
    ```

## 🛠 Tech Stack
-   **Core**: React, Vite
-   **Styling**: Vanilla CSS (Variables, Flexbox, Animations)
-   **Libraries**: TagCloud (for 3D Sphere)
